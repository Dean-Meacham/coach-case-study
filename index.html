project overview.

The objective of this project was to build upon the plethora of existing running apps to design a brand new running app focused on the automated coach feature, giving it more personality, while providing an engaging planned workout experience.

interview insights.

user personas.

the problem identified.

Users lack a clearly defined workout plan that illustrates what to do and when to do it.

final How Might We.

How Might We create an automated running coach that takes all the guesswork out of the user's customized workout plan?

design decisions explained.

A bright color scheme was selected so that it would be vibrant and have an energizing effect on its users. the simple layout translates well between mobile phone screens as well as wearable technology. The size and weight of the text is intended to be legible from a variety of distances.

lessons learned.

This project provided me the opportunity to design an app centered on people who run: wanting to run more effectively, or having a specific race goal. This project was definitely a challenge, but through the process, I was able to refine my critical thinking, research and design skills. As a result, I now have a better understanding of user flow, wireframing and prototyping.
